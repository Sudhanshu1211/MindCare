# Federated Mental Health Diagnosis Chatbot

A modular Python 3.10 project for privacy-preserving, federated mental health risk assessment and diagnosis.

## Modules
- `common/`: Shared utilities (encryption, model loading)
- `chatbot-ui/`: Streamlit chatbot UI
- `mental-health-api/`: FastAPI backend for NLP/diagnosis
- `local-secure-storage/`: Encrypted local data storage
- `federated-learning-node/`: Flower client for federated learning
- `fl-server/`: Federated aggregation server
- `monitor-dashboard/`: Admin dashboard (Streamlit/Dash)

See each module for setup and usage instructions.
